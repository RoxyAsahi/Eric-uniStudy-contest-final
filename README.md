# UniStudy 比赛提交材料

本仓库用于存放 UniStudy 项目的最终参赛提交材料。

## 目录结构

- `在线报名信息/`：项目简介、创作说明、环境说明等报名材料。
- `项目源程序以及安装包/`：UniStudy 源程序、评委运行说明和 Windows 安装包。
- `项目演示视频/`：作品演示视频。
- `诚信承诺书/`：诚信承诺书文件。

## 推荐查看顺序

1. 阅读 `项目源程序以及安装包/评委运行说明.txt`
2. 查看 `在线报名信息/` 中的项目说明材料
3. 观看 `项目演示视频/` 中的演示视频
4. 运行 `项目源程序以及安装包/UniStudy-v0.1.0-Setup.exe`

## 源码运行

源码位于：

```text
项目源程序以及安装包/Unistudy
```

源码目录不包含 `node_modules`。如需从源码运行，请在源码目录执行：

```bash
npm install
npm start
```

或使用锁定依赖版本：

```bash
npm ci
npm start
```

## 说明

安装包和演示视频属于较大的二进制文件，仓库使用 Git LFS 管理此类文件。
